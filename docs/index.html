About Lumena

Lumena is a modern, feature-rich music player designed with Material 3 principles. It offers a seamless listening experience with advanced features like:

  • Synced lyrics support
  • Tag editor
  • Material You dynamic theming
  • YouTube transcripts integration
  • Spotify metadata search
